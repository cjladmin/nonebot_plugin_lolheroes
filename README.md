# LOLHeroes

_✨ 基于NoneBot2实现的 获取LOL英雄背景故事 插件 ✨_

</div>

## 配置说明：

> 1.把下载后的文件夹放在`src/plugins/`路径下

> 2.在`bot.py`文件内添加：`nonebot.load_plugin("lolheroes")`

## 使用说明：

> 直接在群内/私聊发送：
>> lol亚索/疾风剑豪背景故事
